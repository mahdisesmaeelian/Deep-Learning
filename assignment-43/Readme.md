# This projects demonstrates training a Multi Layer Perceptron(MLP) and Convolutional Neural Network (CNN) to classify Mnist and Cifar datasets.
### Result :
| Dataset | MLP (Machine Learning) | CNN + MLP (Deep Learning)
|     :---:      |  :---: | :---:
|Mnist  |97%| 99%|
|Fashion Mnist  |86%| 90%|
|Cifar 10 |36%|68% |
|Cifar 100 |13%|33% |
